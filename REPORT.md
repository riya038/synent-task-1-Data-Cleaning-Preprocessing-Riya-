## Objective

The objective of this project was to clean and preprocess the Titanic dataset by handling missing values, removing duplicates, and preparing the data for analysis and machine learning applications.

---

## Dataset Information

The Titanic dataset contains information about passengers aboard the Titanic, including demographic details, ticket information, and survival status.
