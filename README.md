# Titanic Data Cleaning Project

## 📌 Objective
The objective of this project is to clean and preprocess the Titanic dataset for analysis.

## 📂 Dataset
Titanic Dataset from Kaggle.

## 🛠 Tools & Libraries Used
- Python
- Pandas
- Matplotlib
- Seaborn
- VS Code

## 🔍 Steps Performed
1. Loaded dataset
2. Checked dataset information
3. Handled missing values
4. Removed duplicate rows
5. Renamed columns
6. Saved cleaned dataset
7. Created visualizations

## 📊 Visualizations Created
- Survival Count Plot
- Gender Distribution Plot
- Age Distribution Histogram
- Passenger Class Distribution
- Survival Based on Gender

## 📈 Key Insights
- Female passengers had a higher survival rate
- Most passengers belonged to passenger class 3
- Some columns had missing values which were cleaned

## ✅ Output
Cleaned Titanic dataset ready for analysis.
